# Contributing Guide

`trustcheck` is not accepting pull requests or other code contributions at this time.

In light of software supply-chain incidents such as the XZ Utils backdoor and the Axios backdoor, this project is operating under a solo-developer policy until further notice.

Non-sensitive bugs and feature suggestions may still be reported through
[GitHub Issues](https://github.com/Halfblood-Prince/trustcheck/issues), but pull
requests will not be reviewed or merged.

If you discover a security issue, follow the private reporting guidance in
[SECURITY.md](SECURITY.md).
